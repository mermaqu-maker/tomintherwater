# Tom in the Water — 디자인 레퍼런스 리서치

> 목적: 디자인 시안 착수 전, 저명한 사진작가·아티스트 포트폴리오 홈페이지를 조사해 차용/회피할 패턴 도출.
> 방법: 각 사이트 실제 홈페이지를 fetch 하여 레이아웃·내비·갤러리·컬러·타이포·인터랙션 분석 (2026-07 기준).
> 관련 기획: [01_새사이트_기획_IA.md](./01_새사이트_기획_IA.md)

## 0. 요약 (핵심 발견)

- **수중·다크 무드의 정석은 "다크 배경 + 이미지 전면"**: Zena Holloway, Tim Tadder가 딥톤 배경으로 이미지를 띄우는 방식 → 우리 `#1E1E1E` 방향과 정확히 일치.
- **의외로 순백 배경도 많음**: Christy Lee Rogers, Mark Mawson, Brooke Shaden은 흰 배경 + 이미지 대비. 물 사진의 블루·시안을 살리는 데 흰 배경도 유효하나, 우리는 "시네마틱·명상적" 브랜드 보이스상 다크가 더 맞음.
- **내비게이션은 극도로 절제**: 대부분 5~6개 이하, 로고=홈, 상단 sticky. 우리의 4개 영문 메뉴 방향과 일치.
- **"입장(ENTER)" 게이트·풀블리드 히어로**로 몰입 유도(Erik Johansson, Zena Holloway). 우리 히어로(풀블리드 + 한 줄 카피)와 통함.
- **갤러리 = Masonry 또는 풀폭 블록 + 라이트박스("View fullsize")**가 표준. 우리 Masonry + 감상 전용 라이트박스 결정과 일치.
- **텍스트 최소·타이포 위계 단순**, 캡션은 hover/상세로. 우리 "캡션 없이 hover 오버레이" 결정과 일치.

## 1. 사이트별 분석

관련도: ★★★ = 수중/다크/시네마틱 직접 참고, ★★ = 갤러리 UX 참고, ★ = 구조/전략 참고.

### ★★★ Christy Lee Rogers — https://www.christyleerogers.com/
바로크풍 수중 파인아트. 시네마틱·회화적 무드의 대표주자.
- **레이아웃**: 롱폼 세로 스크롤, 섹션 적층. 그리드보다 **풀폭 이미지 블록 + 서술형 흐름**.
- **내비**: 상·하단 반복(sticky), Home/Collections/Video/Exhibitions + Press·News·About·Contact.
- **갤러리**: 풀폭 블록 + 가로 갤러리, **"View fullsize"** 확대. 프로젝트를 히어로처럼 스포트라이트.
- **컬러/타이포**: 미니멀 화이트 여백, 산세리프, 갤러리 같은 우아함.
- **인터랙션**: "Explore / Reveal More / **Dive Deeper**" 같은 서정적 CTA, back-to-top, 소셜 푸터.
- **차용점**: 프로젝트를 카드가 아니라 **몰입형 스포트라이트**로 다루는 방식, 서정적 CTA 문구 톤.

### ★★★ Zena Holloway — https://zenaholloway.com/
수중 패션·파인아트. **우리와 가장 가까운 무드**(다크 + 수중).
- **레이아웃**: 풀폭 히어로 이미지, 섹션 세로 흐름. 대형 사진 + 짧은 텍스트 + CTA.
- **내비**: 상단 sticky — Water / Root / About / Shop / News / Contact, 로고=홈, 모바일 토글.
- **갤러리**: 프로젝트 카테고리(Earthless/Rivers/Fiction)를 **이미지 블록 + 타이틀 오버레이**로 진입.
- **컬러/무드**: **딥톤 다크 배경**으로 이미지가 튀는 "aquatic realm" 무드. 우리 방향의 실증.
- **타이포**: 큰 헤드라인 + 최소 본문, 갤러리형.
- **차용점**: 다크 배경 + 카테고리 이미지 블록 + 타이틀 오버레이 = 우리 갤러리 hover 오버레이와 직접 연결.

### ★★★ Tim Tadder — https://www.timtadder.com/
볼드한 커머셜/파인아트. **다크 UI의 프리미엄 레퍼런스**.
- **레이아웃**: 상단 + sticky 이중 내비, 대형 히어로 + 그리드 카드.
- **컬러/무드**: **블랙 배경 + 화이트 타이포**의 다크 심미. "Bold and fearless" — 프리미엄·갤러리 분위기.
- **갤러리**: 비주얼 그리드 카드(최소 텍스트 오버레이), 주요 프로젝트는 대형 히어로.
- **차용점**: 다크 배경에서 CTA·텍스트 대비 잡는 법, 그리드 카드 최소 오버레이.

### ★★★ Mark Mawson — https://www.markmawson.com/
수중 'Aqueous' 시리즈 + 리퀴드 커머셜.
- **레이아웃**: 미니멀, 상단 내비(Overview/Portfolios/Motion/Contact/Buy Prints).
- **갤러리**: **Masonry/그리드 + "View fullsize" 라이트박스**. 우리 Masonry+라이트박스와 동일 패턴.
- **컬러/무드**: 화이트 배경, 절제된 럭셔리. "let the work speak".
- **차용점**: Masonry + 라이트박스 표준 구현, Motion(영상) 섹션 분리.

### ★★★ Andreas Franke — http://thesinkingworld.com/
난파선 수중 전시("The Sinking World") 등 초현실 수중 프로젝트.
- **성격**: 프로젝트 단위 마이크로사이트/전시 중심. 스토리텔링·환경 메시지.
- **차용점**: 프로젝트를 **전시/시리즈 단위 서사**로 묶는 접근(우리 프로젝트 상세의 서사화 참고).

### ★★ Erik Johansson — https://www.erikjo.com/
초현실 합성 사진. 시네마틱·미스터리.
- **레이아웃**: **미니멀 센터드 랜딩 + "ENTER" 게이트** → 포트폴리오 별도. 이름 타이포가 지배.
- **무드**: 절제·명상적·큐레이토리얼. 진입 전 기대감 조성.
- **차용점**: 히어로에서 **한 컷 + 큰 타이포 + 진입 유도**로 몰입 만드는 법(우리 히어로 강화 아이디어).

### ★★ Brooke Shaden — https://brookeshaden.com/portfolio
다크·초현실 파인아트 셀프포트레이트.
- **갤러리**: **반응형 Masonry 그리드**, 썸네일 → 풀스크린(타이틀+연도). 15년+ 작업을 일관된 리듬으로.
- **내비**: 이중 내비 — Gallery(Portfolio/Series/Writing)/About/Purchase/Work With Me.
- **컬러**: 따뜻한 뉴트럴/라이트 배경(주제는 어둡지만 프레임은 밝음).
- **차용점**: Masonry + 썸네일→풀스크린(타이틀·연도) = 우리 상세/라이트박스 흐름과 동일.

### ★★ Steve McCurry — https://www.stevemccurry.com/
다큐멘터리 거장. 에디토리얼 큐레이션.
- **레이아웃**: **디렉터리형** — 홈에 이미지 도배보다 주제별 갤러리 링크(Afghanistan/Portraits/Devotion…)로 진입.
- **무드**: 절제·에디토리얼·프리미엄(파인아트 프린트, Selected Works).
- **차용점**: 작업이 많아질 때 **주제/카테고리 진입 구조**로 정리(우리 태그·카테고리 확장 대비).

### ★ Von Wong — https://vonwong.com/
에픽 환경 아트·캠페인.
- **레이아웃**: 내러티브 세로 스크롤, 고정 헤더(Projects/Speaking/Gallery/Blog/Contact).
- **구조**: 프로젝트를 **카드 + 임팩트 지표(노출수·매체·장소)**로 서사화. "8B+ impressions" 등 수치 스토리텔링.
- **컬러**: 모노크롬(블랙 온 화이트), 큰 볼드 타이포.
- **차용점**: 상업(Commercial) 설득에 **성과/신뢰 지표**를 넣는 전략(우리 About·상품 신뢰 강화).

### ★ Chris Burkard — https://www.chrisburkard.com/
어드벤처·자연. (SmugMug 기반 — 빌더 종속)
- **시사점**: 저명 작가도 빌더(SmugMug)를 쓰지만 **커스텀 확장은 제한**. 우리가 Next.js+Supabase로 가는 이유(어드민·좋아요·댓글·의상 카탈로그)를 방증.

## 3. 우리 시안 방향 제언 (리서치 → 디자인)

1. **다크 시네마틱 확정**: `#1E1E1E` 베이스, 무채색 UI, 색은 수중 이미지가 담당. Zena Holloway·Tim Tadder를 1차 무드 레퍼런스로.
2. **히어로는 "한 컷 + 절제된 타이포 + 진입 유도"**: 풀블리드 배경(영상/대표작) + 서정적 한 줄. Erik Johansson의 몰입 진입감을 참고하되 게이트는 과하지 않게.
3. **갤러리 = 다크 배경 위 Masonry + hover 오버레이 + 라이트박스**: Zena(오버레이)·Mawson/Shaden(Masonry+확대) 결합.
4. **프로젝트 상세는 스포트라이트 서사**: Christy Lee Rogers처럼 카드가 아닌 몰입형. 좌 이미지 / 우 설명(이미 결정)에 서정적 크레딧.
5. **타이포 2단 위계 + 서정적 카피**: 큰 헤드라인 + 최소 본문. 영문 세리프/산세 조합 후보를 시안에서 비교.
6. **신뢰 레이어**: About·상품에 경력·클라이언트·성과를 Von Wong식으로 절제 삽입(상업 전환용).
